open fun fetch(filter: TransactionsFilter, options: Options, seriesSelectors: List<SeriesSelector>): Result<List<MenigaTransactionSeries>>

Retrieves a series of transactions aggregated over time.

Return

A list of transaction series objects that match the given criteria

Parameters

filter

A Filter used to filter out the set of transactions which the series aregenerated from. Note: Each has its own filter which allows for selecting asubset of the transactions queried using this filter. Note : The PeriodFromand PeriodTo properties always need to be populated.

options

Option object for the Transaction Series List.

seriesSelectors

A list of SeriesRequests each containing its own filter in order toselect a subset of the "main" set.
